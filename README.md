**My Health Compass** is a modern, AI-powered health assessment tool designed to help individuals evaluate their current daily habits and receive personalized, actionable advice for a healthier lifestyle. Based on the "Health-Promoting Lifestyle Profile" concept, it provides a quick 8-question check-in across multiple health domains.

---

## ✨ Features

- 📝 **8-Question Assessment**: Evaluates key life domains including Stress Management, Social Health, Physical Activity, and more.
- 📊 **Instant Scoring**: Real-time calculation and categorization of your health profile (Poor to Excellent).
- 🤖 **AI Action Plan**: Personalized daily routine recommendations generated by **Gemini 3 Flash** (via Supabase Edge Functions).
- 🎨 **Modern UI**: Clean, responsive, and accessible interface built with Tailwind CSS and Radix UI components.
- 🌗 **Premium Aesthetics**: Smooth transitions, glassmorphism effects, and a serene color palette for a focused experience.

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [Bun](https://bun.sh/)
- A [Supabase](https://supabase.com/) project

### Local Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/apurvakumar17/my-health-compass.git
   cd my-health-compass
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   bun install
   ```

3. **Set up Environment Variables:**
   Create a `.env` file in the root directory and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_supabase_project_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   # or
   bun dev
   ```

5. **Open your browser:**
   Navigate to `http://localhost:8080` (or the port shown in your terminal).
